【できていること】
- 商談中の画面に相手から見えない透過カンペを表示
- 自分・相手の発話をリアルタイムで文字起こし（デュアル）
- ホットキー（Cmd+K / F9）でAIが次の一言を提案
- Auto Coachモード（相手の発話の間に自動でサジェスト）
- AIはGemini / Claude切替可能
- スライドの自動切替検知
- 議事録・要約・スライド画像をローカルに自動保存
- macOS / Windows両対応
- APIキーはOS暗号化領域に保存（セキュア）
- 緊急時にすぐ非表示にできるパニックキー

【動かし方】
1. ZIPダウンロード or git clone
2. Node.js 20以上をインストール
3. npm install
4. APIキーを用意（Gemini必須 / Deepgram推奨 / Anthropic任意）
5. npm run dev で起動 → 設定画面でキー入力

動かしてみてご不明点や修正点あれば気軽に言ってください！
