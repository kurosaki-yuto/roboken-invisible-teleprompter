Mienaq（透過カンペアプリ）の完成版が上がりましたのでご報告と引き渡しをいたします。

【概要】
商談中の Zoom / Google Meet 画面に「相手からは見えない透過カンペ」を重ね、
AI（Gemini / Claude 切替）が次の一言をホットキーで提案するデスクトップアプリです。

【主な機能】
- 自分・相手の発話をリアルタイム文字起こし（Deepgram、デュアル）
- スライド自動切替検知（知覚ハッシュ）
- ホットキー（Cmd+K / F9）で AI 提案
- Auto Coach モード（相手の発話一段落ごとに自動サジェスト）
- 議事録・要約・スライド画像をローカル自動保存
- macOS / Windows 両対応
- API キーは OS 暗号化領域に保存（safeStorage）
- パニックキーで即座に非表示
- B2B（営業/CS/採用面接）トーンに最適化

【リポジトリ】
https://github.com/syun3032-tech/Mienaq

【セットアップ手順（リポジトリ README に詳細あり）】
1. リポジトリから ZIP ダウンロードまたは git clone
2. Node.js LTS（20 以上）をインストール
3. `npm install`
4. API キーを準備
   - Gemini API キー（必須・無料枠あり）
   - Deepgram API キー（推奨・新規 $200 枠）
   - Anthropic API キー（Claude 利用時のみ・任意）
5. `npm run dev` で起動 → 設定画面でキー貼り付け → 接続テスト
