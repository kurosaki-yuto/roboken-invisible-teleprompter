# 会議支援ツールの機能提案

「会議中の最強の武器」＋「会議後の完璧な備忘録」の 2 面構成。
同じデータを活用する表裏一体の機能として実装する。

## 1. 議事メモの自動保存と閲覧

商談中は透明ウィジェットで提案を出しつつ、バックグラウンドで完璧な議事録を自動生成する。

- **タイムライン形式**: 会議終了後にダッシュボードへ自動生成。発言者ごとの文字起こし＋「決定事項」「相手の懸念点」「次回の宿題」を AI が自動で見出し付きで要約
- **検索・閲覧**: 過去の商談をすべてデータベース化。「A 社との前回の打ち合わせ」「〇〇というキーワードが出た商談」を瞬時に検索可能

## 2. スライド認識と完全連動

- **スナップショットの自動保存**: アプリ起動時に「相手の画面共有エリア」を指定しておくと、スライドの切り替わりを検知して自動スクリーンショット
- **資料＋会話のリッチな議事録**: 「このスライドが表示されている時に、相手からこういう質問があり、こう回答した」という画像と会話が完全にリンクした議事録として保存
- **文脈を読んだリアルタイム提案**: スライドに書かれている文字・図表をリアルタイムで AI が読み取り、資料の文脈に沿った助言を透明ウィジェットに表示

## ユーザー操作（3 ステップのみ）

```
1. 会議開始時
   └─ 「Start」ボタンを押し、相手の資料が映る範囲をマウスで四角く囲む

2. 会議中
   └─ 画面上部に浮かぶ「透明なカンペ」を見ながら商談を進める
      （困ったらショートカットキーを押して AI に助けを求める）

3. 会議終了後
   └─ アプリを開けば、スライド画像付きの完璧な議事メモが完成している
```

## SaaS 展開の方向性

- **クローズド専用ツール**: まず自身の PC のみで動くプロトタイプとして開発
- **SaaS 型プロダクト**: 安定後に他メンバーや外部にも提供できる形に拡張
